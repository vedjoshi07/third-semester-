# Netflix Clone

A modern Netflix clone built with HTML, CSS, and JavaScript featuring a responsive design and interactive user interface.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Hero Section**: Featured content with call-to-action buttons
- **Movie Categories**: Multiple rows of movies organized by genre
- **Interactive Modal**: Click on any movie to view details
- **Smooth Animations**: Hover effects and transitions
- **Netflix-like UI**: Authentic Netflix styling and color scheme

## Categories

- Trending Now
- Netflix Originals
- Action Movies
- Comedy Movies
- Horror Movies

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (ES6+)
- Google Fonts

## Getting Started

1. Clone or download the project files
2. Open `index.html` in your web browser
3. Navigate through the different movie categories
4. Click on any movie poster to view more details

## File Structure

```
netflix-clone/
├── index.html          # Main HTML file
├── styles.css          # CSS styles and responsive design
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## Customization

You can easily customize the Netflix clone by:

1. **Adding More Movies**: Edit the `movieLists` object in `script.js`
2. **Changing Colors**: Modify the color variables in `styles.css`
3. **Adding New Categories**: Add new sections to the HTML and corresponding data in JavaScript
4. **Updating Hero Content**: Change the hero section content in `index.html`
